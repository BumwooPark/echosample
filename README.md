# echosample

## Getting Started

Get source
```
$ go get github.com/pangpanglabs/echosample
```

Test
```
$ go test github.com/pangpanglabs/echosample/...
```

Run
```
$ cd $GOPATH/src/github.com/pangpanglabs/echosample
$ go run main.go
```

Visit http://127.0.0.1:8080/
